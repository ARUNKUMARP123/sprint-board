# Sprint 1 Extra Challenge Board

A responsive multi-column board application built using React, Vite, JavaScript, and Bootstrap.

This project was developed for the TIFC2DEV-SAC Sprint 1 Extra Challenges Assessment.

---

## Features

- Multi-column responsive grid layout
- Add columns using + button
- Remove columns using - button
- Set number of columns dynamically
- Drag and drop column reordering
- Move columns left and right
- Edit column titles
- Change column background colors
- Responsive Bootstrap UI

---

## Technologies Used

- React
- Vite
- JavaScript
- Bootstrap
